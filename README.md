HappyHouse
=============
\* Developer : Ayoung Lee, Dongsik Choi

기술 스택
-------------

* 데이터 베이스
  * MySQL

* 외부 API
  * 공공 API
    * 상권 정보
    * 환경점검 시설
    * 아파트 매매 정보
    * 코로나 병원, 보건소
  
  * KAKAO API

* Back-End
  * SpringBoot

* Front-End
  * Quasar
  * VUE.js
